# SASS 란??

- CSS 를 확장하여 유연한 스타일을 작성하는 데 도움이됩니다.
- 변수, 함수, 조건문 등을 사용합니다 ...
- SASS 파일을 모듈로 분할하여 대규모 프로젝트를 위해 CSS를 쉽게 유지할 수있게 해줍니다.

- SASS로 작성하면 브라우저가 코드를 이해하지 못합니다.
- 그것은 CSS로 번역해야합니다.
- SASS는 Ruby로 작성되었습니다.
